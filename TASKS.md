# ЭТАП 15 — Безопасность и Auth

Статус: ✅

Задачи:

✅ Валидация Telegram init data (HMAC-SHA256)

✅ Единая функция authenticateRequest

✅ Поддержка x-user-id для локальной разработки

✅ Поддержка x-telegram-init-data для production

✅ 401 при отсутствии авторизации

Кратко:

Добавлена полноценная валидация Telegram init data через HMAC-SHA256 с bot token. В http.ts внедрён middleware authenticateRequest, который проверяет подпись init data или использует x-user-id для локальной разработки. Без авторизации API возвращает 401. CORS расширен для x-telegram-init-data.
