# ЦЕЛЬ ПРОЕКТА

Создать Telegram Mini App для ежедневной духовной практики.

Это не ежедневник.

Это не Notion.

Это не Obsidian.

Это не CRM.

Это персональный помощник практикующего.

Основная задача приложения — максимально упростить выполнение ежедневной практики.

Приложение должно стать тем инструментом, который пользователь открывает каждый день.

# КОНЦЕПЦИЯ

Каждый день пользователь получает индивидуальное расписание практики.

Практики постоянно меняются.

Например

понедельник

• цигун

• пранаяма

• джапа

• чтение

вторник

• махамудра

• созерцание

• пранаяма

• прогулка

среда

совершенно другой набор.

Приложение не должно предполагать фиксированный список практик.

# ОСНОВНЫЕ РАЗДЕЛЫ

## Сегодня

Главный экран.

Показывает:

• сегодняшние практики

• время выполнения

• прогресс

• быстрый запуск таймера

• статус выполнения

---

## Планировщик

Позволяет составить расписание.

Способы:

• вручную

• текстом

• голосом

ИИ понимает естественный язык.

Примеры.

"Завтра цигун 45 минут, пранаяма 20, джапа 15"

или

"Завтра как сегодня, только вместо пранаямы сделай махамудру."

или

"Добавь вечером ещё чтение."

---

## Библиотека практик

Главный раздел приложения.

Каждая практика хранится один раз.

Структура практики:

Название

Описание

Категория

Картинка

Иконка

Продолжительность по умолчанию

Цвет

История выполнения

Связанные материалы

Личные заметки

Практики могут быть:

созданы

отредактированы

удалены

переименованы

пользователем.

Никаких фиксированных практик.
